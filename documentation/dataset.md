# Dataset

**Note on Currency Formatting:** All currency values in this dataset follow South African formatting conventions (en-ZA locale):
- **Thousand separator:** Non-breaking space (shown as regular space in markdown)
- **Decimal separator:** Comma (,)
- **Format example:** R 1 234 567,89

For readability in markdown tables, some values may display with period (.) as decimal separator, but the application will format all currency using proper en-ZA locale with comma decimal separators.
